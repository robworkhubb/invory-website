## SPECIFICAZIONI TECNICHE
- Formato: PNG per screenshot, JPG per foto, ICO per favicon
- Ottimizzazione: Compressi per il web (max 500KB per immagine)
- Alt text: Includere descrizioni SEO-friendly
- Responsive: Versioni per mobile se necessario

## SUGGERIMENTI PER LE IMMAGINI
1. hero-app.png: Mostrare l'interfaccia principale dell'app con dati realistici
2. app-demo.png: Screenshot di una funzionalità specifica (es. dashboard, gestione scorte)
3. og-image.jpg: Design accattivante con logo Invory e call-to-action
4. twitter-image.jpg: Versione ottimizzata per Twitter con testo leggibile

## OTTIMIZZAZIONE SEO
- Nomi file descrittivi (es. "invory-dashboard-screenshot.png")
- Alt text con parole chiave target
- Dimensioni appropriate per ogni dispositivo
- Compressione senza perdita di qualità visiva
